Que es HTML?

Actualmente la mayoria de las personas han visto cientos de paginas web pero mucho se preguntaran ¿Qué es HTML? ¿Para que me sirve?

Si buscas en Google o en cualquier otro buscador la definición de HTML, probablemente escontrarás esta: " HTML es un lenguaje de programación que se utiliza para el desarrollo de páginas de internet. Se trata de las siglas que corresponden HiperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto"

¿Para que sirve?

HTML sirve para codificar la estructura de una página web. Para ello se vale de las etiquetas o tags. Los navegadores no muestran las etiquetas pero las utilizan para representar el contenido de una página.
